# Usage
```
$ chip8_emulator your_rom
```
## Optional arguments 
-d  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Enables debug mode (it only displays the opcodes)  
-fps *value*  
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Runs the program at *value* fps.  
```
$ chip8_emulator your_rom -d -fps 600
```
